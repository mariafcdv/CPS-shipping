# CPS Shipping - Cotizador de envíos

Prueba técnica: Cotizador de envíos CPS (Angular + Spring Boot + SQL Server).

## Fórmula
(peso * tarifa) + 1.66 * alto * largo * ancho – descuento_cliente * 0.5 * peso

## Estructura
- Backend/ (Spring Boot REST API)
- DB/ (script SQL Server)
- Frontend/ (Angular)

## Base de datos
Script de creación: `DB/CPS_shipping.sql`
